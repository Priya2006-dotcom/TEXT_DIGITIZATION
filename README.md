# TEXT_DIGITIZATION

## Overview
TEXT_DIGITIZATION is an AI-powered handwritten text digitization system that converts handwritten notes into editable digital text using Optical Character Recognition (OCR) and Natural Language Processing (NLP).

The system helps students, teachers, and organizations digitize handwritten documents efficiently and accurately.

## Features
- Upload handwritten note images
- Extract text using OCR
- Convert handwritten text into digital format
- Display extracted text instantly
- User-friendly interface
- Fast and efficient text processing

## Technologies Used
- Python
- Streamlit
- OCR (EasyOCR / Tesseract OCR)
- OpenCV
- NumPy
- NLP Techniques

## System Workflow

1. Upload handwritten image
2. Preprocess image
3. Extract text using OCR
4. Convert to digital text
5. Display output to user

## Project Architecture

User
↓
Upload Image
↓
Image Preprocessing
↓
OCR Engine
↓
Text Extraction
↓
Digital Text Output
